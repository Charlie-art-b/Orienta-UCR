## Career Guidance System
📌 Overview

This project is a full-stack web application designed to help students identify the career path that best fits their skills and interests.

The system provides personalized guidance through a structured platform that includes role-based access, notifications, and modular data management.

🧠 Architecture

The application follows a client-server architecture:

Backend:

Spring Boot (Java)
Handles business logic, authentication, and data management.

JPA (Java Persistence API)
Used for ORM and database interactions.

JWT Authentication
Implements secure role-based access control (roles & permissions).

Frontend:

React.js
Dynamic and responsive user interface.

Atomic Design Pattern
Component-based architecture for scalability and reusability.

Bootstrap
Responsive styling and UI components.

⚙️ Core Features

🎯 Career recommendation system for students

🔐 Authentication and authorization using JWT

👥 Role and permission management

🔔 Notification system

🗂️ Modular CRUD operations across multiple entities

📱 Responsive UI with Bootstrap

🧩 Scalable frontend using Atomic Design

🛠️ Technologies Used

Backend: Spring Boot (Java)

Frontend: React.js

Architecture: Atomic Design (Frontend)

Authentication: JWT

Database Access: JPA / Hibernate

Styling: Bootstrap
